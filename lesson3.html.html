Lesson 3: The Computing System

1. Evolution

This section tracks the progression of computing tools from ancient instruments to the birth of modern computing.

  • Ancient Tools: The Abacus used for arithmetic and the Astrolabe for celestial navigation and latitude.
  • The Blueprint: Charles Babbage’s Analytical Engine—the first design featuring a store (memory), mill (processor), reader (input), and printer (output).
  • The Electronic Age: The ENIAC (1943), designed for artillery tables, which paved the way for the Association for Computing Machinery (ACM).
  • Pioneers: Visionaries like Alan Turing (Theory), Grace Hopper (COBOL), and the founders of Apple and Microsoft.
2. Classification

Modern computers are categorized based on their size, power, and intended use.

  • Microcomputers: Devices for individual use, including desktops, laptops, tablets, and wearables.
  • Mainframes: Large-scale institutional computers that handle massive databases and enterprise applications for banks and governments.
  • Supercomputers: The most powerful tier, used for complex simulations like weather forecasting and quantum mechanics.
3. Architecture

A computer system is a synergy of Hardware (tangible) and Software (intangible) focused on the Input-Process-Output cycle.

  • The CPU (The Brain): Composed of the Memory Unit (temporary storage), Control Unit (CU) which directs data flow, and the Arithmetic Logic Unit (ALU).
  • ALU Functions: Performs mathematical and logical operations such as AND, OR, and NOT.
  • I/O Devices: Bridge the gap between human instructions and machine-readable data.
4. Operations

Explains the internal logic of how data is handled and stored.

  • Machine Cycle: The four-step process of Fetch (get data), Decode (interpret), Execute (process), and Store (save).
  • Memory Hierarchy:
    • RAM: Volatile, high-speed primary memory.
    • ROM: Non-volatile memory for permanent instructions.
    • Cache: Ultra-fast access for the processor.
    • Secondary Storage: Long-term storage like Hard Drives.
  • Software: Includes System Software (OS, drivers, translators) and utility programs.
